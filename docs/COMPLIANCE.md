# 连山同城 LsLife · 商业化合规与上线清单（中国大陆）

本清单覆盖 App 上线前的资质、隐私合规、支付、内容与应用市场要求。带 [外部] 的项需企业资质或第三方账号，无法在代码层完成。

## 1. 主体与资质 [外部]
- [ ] 企业营业执照（经营范围含互联网/信息服务）
- [ ] ICP 备案（App/域名）；如涉及经营性信息服务，办理 ICP 经营许可证（增值电信业务）
- [ ] 若含即时配送/本地生活撮合，核实是否需要相关经营许可
- [ ] 软件著作权登记（部分应用市场要求）
- [ ] 网络安全等级保护测评（等保 2.0，通常二级起）

## 2. 支付合规 [外部]
- [ ] 资金必须通过持牌第三方（微信支付/支付宝/银联），平台不得自建资金池
- [ ] 申请微信支付商户号、支付宝应用与签约
- [ ] 服务端统一下单 + 异步回调验签 + 每日对账（见 `backend/src/services/payment.ts`）
- [ ] 退款、投诉、资金安全风控流程

## 3. 实名与内容安全
- [x] 用户实名认证接口（`/api/auth/realname`，身份证仅存哈希）
- [x] UGC 发布机审（敏感词，见 `services/moderation.ts`）
- [ ] [外部] 接入云内容安全（文本/图片/涉政涉黄）+ 人工审核队列
- [ ] 违规内容与日志留存（依法留存期限）
- [ ] 未成年人保护与防沉迷（如涉及）

## 4. 个人信息保护（个保法/App 收集使用规范）
- [x] 权限最小化：`AndroidManifest.xml` 仅申请 网络/定位/相机/通知，运行时动态申请
- [ ] 首次启动隐私政策弹窗（同意前不收集）、二次确认
- [ ] 个人信息收集清单（收集项、目的、频率、第三方共享）
- [ ] 第三方 SDK 目录与合规说明（地图、支付、推送、统计）
- [ ] 账号注销、数据导出、撤回同意入口
- [ ] 敏感权限（定位/相机）使用场景说明

## 5. 客户端安全加固
- [ ] release 混淆/资源压缩（已开启 `isMinifyEnabled`）
- [ ] 网络强制 HTTPS（release 已用 https/wss；关闭 `usesCleartextTraffic`）
- [ ] 证书校验/防抓包（关键接口）、本地敏感数据加密（DataStore/Keystore）
- [ ] 加固与渠道签名校验

## 6. 应用市场上架 [外部]
- [ ] 华为、小米、OPPO、vivo、荣耀、应用宝多渠道账号与开发者认证
- [ ] 各市场隐私合规检测（隐私政策、权限、SDK）
- [ ] 软著/资质材料、应用分级、截图与描述
- [ ] 多渠道打包（productFlavors）与渠道统计

## 7. 可观测与运维
